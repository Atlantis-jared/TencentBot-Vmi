# TencentBot-vmi 专属高帧率 Web 监控流 (DXGI Streamer)

如果你被 PVE Web VNC 的卡顿困扰，又为了避免罗技外挂失效而**不敢使用 RDP**连接，这个官方配套的小工具可以完美解决你的痛点。

它同样使用了与 `TencentBot-vmi` 相同的核心底层技术——**Windows DXGI Desktop Duplication API** 来抓取显卡里的原生画面，然后不经过 CPU 重编解码消耗，直接封装成 MJPEG 视频流，通过内网 HTTP 广播。

## 原理优势
1. **纯只播不控**：你只会在网页里“看直播”，物理鼠标操作绝对不会传进虚拟机，100% 杜绝你手欠干扰后台机器人的任何模拟点击。
2. **极低消耗 & 高帧率**：底层直接拦截显存，在内网你可以享受到接近 60 帧无撕裂的丝滑画面监控。
3. **保持真机 Console 对话**：只要你在 PVE 控制台登录进了桌面，就不用再去碰它。在局域网的任意手机、电脑打开网页，就能实时监工。

## 运行环境

这只是一个单独的 Python 脚本，需在你的 **Windows 虚拟机内部**（运行外挂和游戏的那台机器）执行：

1. 确保虚拟机里装了 Python3。
2. 打开虚拟机的 cmd / PowerShell 进入当前目录，安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 运行服务：
   ```bash
   python streamer.py
   ```

## 观看画面

运行成功后，终端会提示 `监听 0.0.0.0:5000`。
回到你自己的物理电脑（或手机）。在浏览器里输入虚拟机的局域网 IP 加入 `5000` 端口即可！
例如：`http://192.168.1.100:5000`

此时浏览器就会显示实时超高帧率的游戏画面，且再也不用担心外挂掉线了。
